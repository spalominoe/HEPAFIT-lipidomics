# HEPAFIT Lipidomics

**Lipidomic analysis of serum from adolescents with overweight/obesity submitted to different doses of exercise: A secondary analysis of HEPAFIT Study**

### Authors
Robinson Ramírez-Vélez, Sara Palomino, Katherine González-Ruíz, Jorge Enrique Correa-Bautista, Mikel Izquierdo

### About This Repository
This repository contains the analytical code and outputs for the HEPAFIT secondary analysis study. 
It includes:
- R scripts used for statistical analysis and bioinformatics processing
- Data visualization plots
- Summary tables

### Data Availability
The raw lipidomics data is not hosted publicly in this repository. For data access requests, please contact Robin or Sara directly.
